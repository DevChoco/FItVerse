# FItVerse
> - FIt_Verse_App
> - 사용자의 신체에 맞춘 옷(Fit)과 가상 세계(Universe)를 합쳐, 개개인의 맞춤 패션 경험을 제공하는 의미한다ㅎㅎ
> - 목표 : 사용자의 퍼스널컬러, 체형 등의 데이터를 분석해 알맞은 옷을 추천해주고 해당 옷의 예상 착용 사진도 제공함..
